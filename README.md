# Pharmacy Sales & Workforce Optimization Dashboard

## Project Overview
This project transforms raw pharmaceutical sales data into an actionable business intelligence tool. It utilizes a **Star Schema** to analyze medication dispensing patterns across different ATC classifications.

## Key Features
*   **Time Intelligence:** Custom DAX measures for Month-over-Month (MoM) growth using a dedicated Calendar Dimension.
*   **Operational Insights:** Hourly sales intensity tracking to optimize pharmacy staffing levels.
*   **Data Modeling:** Implementation of a Star Schema to handle high-granularity (hourly) data while maintaining performance.

## Technical Stack
*   **Power BI**: Data visualization and modeling.
*   **DAX**: Advanced measures for growth and frequency analysis.
*   **Power Query**: Data cleaning and normalization of drug categories.

## How to Use
1. Download the `.pbix` file from the `/pbix` folder.
2. Open in Power BI Desktop.
3. Use the **Shift Slicer** to view performance during Morning vs. Night shifts.
